CSS Snippet Cheatsheet

Ever have trouble realling the exact syntax for your favorite CSS code? Give it a permanent home and add it to this page! Select any snippert below and it'll automatically select all of the code for you to copy.

Flexbox Row

Use these three properties to create a Flexbox row layout.


        .row {
          display:flex;
          flex-direction: row;
          flex-wrap: wrap;
      }

Flexbox Column

Use this to create a Flexbod column layout.


        .column {
            display: flex;
            flex-direction: column;
      }

CSS Grid Layout

Build a 12-column layout using CSS Grid 


          .grid {
              display:grid;
              width: 100%;
              grid-template-columns: repeat(12, 1fr);
          }

Linear Gradients

This will create a background linear gradient from top to buttom.


          .linear-gradient-background {
              background-image: linear-gradient(
                  rgba(232, 102, 236, 0.3) 0%,
                  rgba(232,102,236,0.6) 100%
              );
          }

Box Transition Glow

Use transition and box shadows to glow on hover.


         .code-card .card-header {
             border-radius: 8px;
             transition: all 0.5s ease-in-out;
         }
